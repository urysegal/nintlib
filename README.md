# nintlib
Multi-dimensional quadrature by John Burkardt


This code based on https://people.math.sc.edu/Burkardt/c_src/nintlib/nintlib.html and repurposed to calculate various Slater-Type-Orbital-based integrals used in computational chemistry.

Reference:

Philip Davis, Philip Rabinowitz, Methods of Numerical Integration,Second Edition,Dover, 2007,ISBN: 0486453391, LC: QA299.3.D28.
